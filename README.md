### Compile and run Java program in Conmand Prompt

1. Find the path of java files
```
cd C:/RRCworkspace/Programming1
```

2. Complie Java code, and the FileName.class will be created.
```
javac FileName.java
```

3. Run the Java program
```
java FileName
```
